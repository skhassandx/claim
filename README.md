# My Robi App - Daily Point Claim Automation

এই প্রজেক্টটি ব্যবহার করে আপনি টার্মুক্স (Termux) বা গিটহাব অ্যাকশন (GitHub Actions) এর মাধ্যমে রবি অ্যাপের ডেইলি রিওয়ার্ড পয়েন্ট স্বয়ংক্রিয়ভাবে ক্লেইম করতে পারবেন।

## 🚀 টার্মুক্স (Termux) এ সেটআপ করার নিয়ম

নিচের কমান্ডগুলো এক এক করে বা একসাথে কপি করে আপনার টার্মুক্সে পেস্ট করুন:

```bash
# ১. স্টোরেজ পারমিশন দিন (যদি আগে না দিয়ে থাকেন)
termux-setup-storage

# ২. প্যাকেজগুলো আপডেট করুন
pkg update -y && pkg upgrade -y

# ৩. গিট (Git) এবং পাইথন (Python) ইন্সটল করুন
pkg install git python -y

# ৪. রিপোজিটরি ক্লোন করুন
git clone https://github.com/skhassandx/claim.git

# ৫. ডিরেক্টরিতে প্রবেশ করুন
cd claim

# ৬. প্রয়োজনীয় রিকোয়ারমেন্টস ইন্সটল করুন
pip install -r requirements.txt

# ৭. স্ক্রিপ্টটি রান করুন
python robi_claim.py
```

## ⚠️ গুরুত্বপূর্ণ নোট (JWT Token)
আপনার `robi_claim.py` ফাইলে যে `Authorization` (Bearer Token) ব্যবহার করা হয়েছে, তার একটি নির্দিষ্ট মেয়াদ আছে। মেয়াদ শেষ হয়ে গেলে স্ক্রিপ্টটি আর কাজ করবে না এবং `401 Token Expired` এরর দেখাবে। তখন আপনাকে পুনরায় প্যাকেট ক্যাপচারিং অ্যাপ দিয়ে রবি অ্যাপ থেকে নতুন টোকেন বের করে ফাইলে আপডেট করতে হবে।
